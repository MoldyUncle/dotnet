# BLACK-JACK (Oko bere)

Jednoduše vytvořena hra Black-jack do počítačové konzole, kde hráč hraje proti počítači

## Funkce hry

* **HIT**: Hráč si může brát karty
* **STAND**: Hráč může stát a, už si nebrat
* **PROTI HRÁČ**: Hráč hraje proti počítači a snaží se vyhrát
* **BARVY**: Hra je barevně zpracovaná aby byla pŕehledná
* **ZÁZNAM**: Výhry a prohry se ukládají i po ukončení hry 

## Pravidla hry

* Cílem je se dostat co nejblíž k číslu 21 bez toho aniž by bylo přesaženo
* Hráč si může vzít náhodnou kartu od 1 do 11
* Jestli hráč už nechce hrát může stát a předat hru protivníkovi
* Protivník si bere karty do té doby do kdy má 17 a více
